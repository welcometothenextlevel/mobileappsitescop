# SITESCOP Inspection Platform Demo

Self-contained clickable product demo for the SITESCOP building and pest inspection platform.

## Run locally

Open `index.html` directly, or serve this folder with any static web server.

The login credentials are pre-filled. The demo includes responsive desktop/mobile layouts, jobs, job creation, building and pest inspection forms, reusable checklist options, photo placeholders, report preview, recycle bin, and company settings.

This is a front-end approval prototype. Authentication, cloud storage, offline synchronization, real photo capture, email, and PDF generation are represented as interactive placeholders for the production phase.
